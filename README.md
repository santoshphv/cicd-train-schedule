## Problem Statement
Automate the build and deployment of a train schedule application using CI/CD pipeline.

## Tools Used
- GitHub
- Jenkins
- Docker
- Docker Hub
- Kubernetes
- Prometheus
- Grafana

## CI/CD Flow
1. Developer pushes code to GitHub
2. Jenkins triggers CI pipeline
3. Docker image is built
4. Image is pushed to Docker Hub
5. Kubernetes deploys the application
6. Prometheus monitors metrics
7. Grafana visualizes dashboards

## Benefits
- Automated deployment
- Faster releases
- Reduced manual errors
- Continuous monitoring
